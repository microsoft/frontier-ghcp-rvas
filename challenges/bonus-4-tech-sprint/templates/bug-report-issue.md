# Bug Report Template

Use this structure when filing bugs as GitHub Issues.

---

**Title format:** `[Bug] Short description of the problem`

**Body:**

```markdown
## Summary

One-sentence description of the bug.

## Steps to Reproduce

1. Go to ...
2. Click on ...
3. Observe ...

## Expected Behavior

What should happen.

## Actual Behavior

What actually happens.

## Screenshot / Error

Paste screenshots or error messages here.

## Environment

- Browser:
- OS:
- Local / Deployed:

## Priority

Blocker / High / Medium / Low
```

**Labels to apply:**

- Type: `bug`
- Priority: `priority:blocker`, `priority:high`, `priority:medium`, `priority:low`
- Sprint: `sprint-1` or `sprint-2`
