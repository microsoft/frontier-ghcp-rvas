# GitHub Copilot Enterprise Hackathon

This site publishes the repo's existing markdown guides through MkDocs so participants can move from track selection to setup to the challenge material in a fixed order.

Start here:

1. Open [Challenge Selection](challenge-selection.md) to pick the right path.
2. Complete [Getting Started](tracks/getting-started.md) before opening a challenge.
3. Use the [Track Catalog](tracks/README.md) or jump straight into a track guide from the navigation.

Coaches can also open the [Challenge Set Builder](challenge-builder.md) to share a smaller URL-only set for a workshop.

The navigation keeps the core tracks first, then the challenge 7-21 tracks, then the shared reference material used across the hackathon.
